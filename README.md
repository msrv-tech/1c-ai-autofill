# 1C AI Autofill

Расширение для **1С:Предприятие 8.3** (BSL) для автоматической генерации **продающих описаний товара** и автозаполнения **дополнительных реквизитов/характеристик номенклатуры** с помощью AI через прокси‑сервис **Gitsell**.

**Ключевые слова (для поиска):** 1С, 1C:Enterprise, 1C Enterprise 8, 1С:Предприятие 8.3, BSL, 1С расширение (`.cfe`), управляемые формы, номенклатура, карточка товара, интернет‑магазин, маркетплейсы, УНФ, Розница, УТ11, КА, ERP, автозаполнение, AI, GPT, GigaChat, YandexGPT, OpenAI.

## Область применения
Данное расширение будет полезно всем владельцам интернет-магазинов, каталога товаров и продающим через маркетплейсы. Расширение создает уникальное продающее описание товара по его наименованию, а также может заполнять дополнительные реквизиты (характеристики).

## Технические требования

### Совместимость
Расширение адаптировано для основных конфигураций на управляемых формах: 
*   Управление нашей фирмой (УНФ)
*   1С:Розница
*   Управление торговлей 11 (УТ)
*   Комплексная автоматизация (КА)
*   1С:ERP Управление предприятием

Для установки выберите подходящую версию в разделе [Releases](https://github.com/msrv-tech/1c-ai-autofill/releases):
*   **`UNF_Rozn`** — для УНФ и Розницы.
*   **`UT_KA_ERP`** — для УТ, КА и ERP.

### Технические особенности
Работу расширения обеспечивает API сервиса [Gitsell](https://gitsell.ru).
*   Для работы **не требуется VPN**.
*   Поддерживаются различные модели ИИ (GigaChat, YandexGPT, OpenAI через прокси и др.).
*   Требуется регистрация на сервисе и получение API-ключа.

## Установка и настройка

1.  Скачайте файл расширения (`.cfe`) из [последнего релиза](https://github.com/msrv-tech/1c-ai-autofill/releases).
2.  Установите расширение в конфигураторе или в режиме Предприятия ("Администрирование" -> "Печатные формы, отчеты и обработки" -> "Расширения").
3.  Снимите галку "Безопасный режим" (для возможности выполнения запросов к API).
4.  Перезапустите сеанс 1С.
5.  В карточке номенклатуры нажмите на ссылку **"Настройки" (⚙)**.
6.  Укажите ваш API-токен (полученный в Gitsell) и выберите модель.

## Использование

В карточке номенклатуры появятся ссылки:
*   **"Заполнить ИИ"** — генерирует описание товара на основе наименования.
*   **"⚙"** — открывает настройки.

В настройках также можно включить и настроить **автозаполнение дополнительных реквизитов**.

## Примеры работы расширения

- **Запрос:** «Автошина Cordiant Road Runner 195/65R15 91H»
  **Ответ:** «Автошина Cordiant Road Runner 195/65R15 91H - надежный выбор для тех, кто ценит комфорт и безопасность на дороге. Ее высокотехнологичный дизайн исключает скольжение на мокрой поверхности и обеспечивает стабильность при движении на трассе...»

- **Запрос:** «Ведро с отжимом»
  **Ответ:** «Обновите свой домашний инвентарь с нашим беспроигрышным ведром с отжимом! Идеальное решение для уборки дома, на даче или в любом домашнем хозяйстве...»

- **Запрос:** «Chanel №5 Парфюмерная вода, 100 мл»
  **Ответ:** «Chanel №5 - это непревзойденный аромат, который известен своей богатой историей и безупречным качеством. Эта парфюмерная вода создана легендарным дизайнером Коко Шанель...»

## English (short)
