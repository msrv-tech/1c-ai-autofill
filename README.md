# 1C AI Autofill

Расширение для автоматического заполнения описаний и дополнительных реквизитов номенклатуры с помощью AI (Gitsell).

## Ветки

*   **`UNF_Rozn`** — версия для УНФ и Розницы.
*   **`UT_KA_ERP`** — версия для УТ, КА и ERP.

## Установка

Скачайте актуальный релиз (`.cfe`) из раздела [Releases](https://github.com/msrv-tech/1c-ai-autofill/releases).
