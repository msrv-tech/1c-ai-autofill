# 1C AI Autofill

## Версия для 1С:Фреш (`UNF_Rozn_Fresh`) — что изменено/ограничено

- **Отключены обновления/скачивание с GitHub**: функционал проверки обновлений и загрузки/обновления расширения через GitHub в версии для 1С:Фреш убран.
- **Нет GitHub релизов для Fresh**: теги/релизы на GitHub публикуются для “обычных” сборок (например, `UNF_Rozn_*`), а для ветки **Fresh релизы не публикуются**.
- **Обновление только через сервис**: установка/обновление выполняются в **1С:Фреш** и **1С:Плюс** через менеджер сервиса.
- **Сервер прокси фиксированный**: реквизит настроек `АдресСервера` остаётся в форме, но **не используется** — обращения выполняются только к **`gitsell.ru`**.
