# 1C AI Autofill

UPD 16.11.23 Для работы API требуется VPN или прокси сервер

С 16.11.23 действуют ограничения для использования API из стран, доступных в списке [Supported countries](https://platform.openai.com/docs/supported-countries). Среди них, к сожалению, нет РФ, поэтому для работы возможны 3 варианта:

- Постоянно включенный VPN.
- Создание прокси сервера (например, по отдельной инструкции).
- Переход на аналогичное отечественное решение от GigaChat (Сбер).

## Область применения
Данное расширение будет полезно всем владельцам интернет-магазинов, каталога товаров и продающим через маркетплейсы. Расширение создает уникальное продающее описание товара по его наименованию.

## Технические требования

### Совместимость
Расширение адаптировано для основных конфигураций на управляемых формах: Управление торговлей 11, Комплексная автоматизация 2.5, Управление нашей фирмой 3, 1С:ERP Управление предприятием 2.5. Если вам необходимо расширение для вашей конфигурации на управляемых формах — пожалуйста, свяжитесь с технической поддержкой. Расширение прошло аудит в сервисе 1cfresh.com. Код расширения полностью открыт.

### Технические ограничения
Работу расширения обеспечивает REST API от chat.openai.com. ВНИМАНИЕ: сервис платный. Стоимость указана на странице [openai.com/pricing](https://openai.com/pricing) (см. модель gpt-3.5-turbo). При первичной регистрации дается $5 для тестов в зависимости от частоты использования. Этого хватит примерно на 3000 описаний номенклатуры. Тестовый период заканчивается в течение текущего месяца, после чего необходимо перейти на платный тариф. Статистика по использованию: `https://platform.openai.com/account/billing/overview`.

## Регистрация
На данный момент регистрация возможна по ссылке `https://chat.openai.com/chat` только с использованием VPN. Также вам понадобится не российский номер телефона. Можно использовать eSIM или сервисы аренды номера (например, `https://sms-activate.org/ru/`). Это необходимо только для регистрации и получения API key (для работы с сервисом далее VPN не нужен).

4) Получаем свой API key: `https://platform.openai.com/account/api-keys` (VPN всё ещё нужен).

Скопируйте свой API key в настройки 1С.

5) Теперь расширение может получать описание товаров. В карточке номенклатуры есть ссылка «Заполнить из ChatGPT». В запросе передается наименование товара, а результат возвращается в поле «Описание».

## Примеры работы расширения
Расширение будет полезно всем интернет-магазинам, а также тем, кто работает с маркетплейсами. Несколько примеров:

  Ответ: «Автошина Cordiant Road Runner 195/65R15 91H — надежный выбор для тех, кто ценит комфорт и безопасность на дороге. Ее высокотехнологичный дизайн исключает скольжение на мокрой поверхности и обеспечивает стабильность при движении на трассе. Широкие плечевые блоки и мягкий рисунок протектора гарантируют превосходную управляемость и долговечность…»

- Запрос: «Ведро с отжимом»
  
  Ответ: «Обновите свой домашний инвентарь с нашим беспроигрышным ведром с отжимом! Идеальное решение для уборки дома, на даче или в любом хозяйстве…»
